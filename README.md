# Todo App (React + TypeScript + Bootstrap)

Онлайн-приложение списка задач с авторизацией по имени пользователя и хранением данных в `localStorage`.

## Функционал

- Авторизация: ввод имени пользователя в начале приложения
- Хранение задач в `localStorage` отдельно для каждого пользователя
- Рендер задач после входа
- Добавление задачи:
  - по нажатию `Enter`
  - по клику на кнопку `Сохранить`
  - кнопка `Сохранить` отключена, пока поле ввода пустое (и при загрузке тоже)
  - после добавления поле очищается
- Управление задачами:
  - завершение задачи (вкл/выкл)
  - удаление с подтверждением через модальное окно
  - редактирование через модальное окно
- Приоритет задачи:
  - обычная (зелёный)
  - важная (жёлтый)
  - срочная (красный)
- Адаптивный интерфейс (desktop/tablet/mobile)
- Минималистичный чёрно-серо-белый дизайн с анимациями

## Стек

- React
- TypeScript
- Bootstrap 5 (+ `react-bootstrap`)
- Webpack + `ts-loader`

## Установка и запуск

```bash
npm install
npm run dev
```

Откройте в браузере: `http://localhost:5173/`

## Сборка

```bash
npm run build
```

Собранный проект попадает в `dist/`.

## GitHub Pages (gh-pages)

Настроен GitHub Actions workflow, который после сборки публикует содержимое `dist/` в ветку `gh-pages`.

Workflow находится по пути:
- `.github/workflows/gh-pages.yml`

После пуша в ветку `main` сайт будет доступен по адресу вида:
- `https://<имя_пользователя>.github.io/<имя_репозитория>/`
